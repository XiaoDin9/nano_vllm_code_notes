Sequence
token0 ................................ token699

     cached                compute
---------------|---------------------------
0             256                         700

Query: [256~699]
Key:   [0~699]

KV Cache:
    Logical blocks: block0       block1       block2
    Physical blocks:  7            15           3
                    cached        write        write
    slot_mapping:               3840~4095     768~955

FlashAttention:
    Q = new tokens
    K = all tokens

Attention:
    new token attends all history
